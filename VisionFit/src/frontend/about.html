About This Project

EyewearAI is a smart, AI-powered web system designed to help users choose the most suitable eyewear based on their unique face shape. It combines facial landmark detection, face shape classification, rule-based recommendations, and augmented reality (AR) for a modern virtual try-on experience.

The project was developed as part of a Final Year Project for the Bachelor of Computer Science (Artificial Intelligence) program at Universiti Teknologi MARA (UiTM).

Key technologies used include:

  • Python (Flask) – for backend processing
  • HTML/CSS/JavaScript – for frontend interaction
  • Computer Vision (OpenCV, MTCNN)
  • Machine Learning (CNN for face shape classification)
  • Augmented Reality using WebAR or overlay with JS

The goal is to provide a seamless and accessible eyewear recommendation system — especially useful for users shopping online or virtually.
